# HLRightSlipMenu
## 一个非常轻巧的右滑菜单，一句话搞定右滑菜单
### 使用方法直接调用下面的方法即可实现   
[[SHLRightSlipMenu shareInstance] addLeftview:leftview toviewctrl:self.tabBarController];
#### 核心代码通过手势获取触摸点来判断左滑和右滑。
#### 用到的主要实现类为SHLRightSlipMenu。
#### 实现了阴影的渐变效果
